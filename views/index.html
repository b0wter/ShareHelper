Share Helper

This website allows you to input urls given by your streaming and convert them into links for other streaming services.

Heres a quick example, you're using Spotify but one of your friends has a Deezer subscription. Every time you want to share a track with your friend you have to tell him/her the name of the track and artist. With this service you take the link and it is automatically turned into a link for several other services.

Simply paste the link generated from your streaming app (Spotify, Deezer, ...) above.

The webpage does not yet offer an api but you can instantly get to the result page if you send a "POST"-request to this url with the "x-www-form-urlencoded" parameter "sharedUrl" and the url you want to convert as value. This can be useful for the shortcut app on iOS.

Alternatively, you can simply call this url with the query parameter "sharedUrl" and an url encoded link, e.g.:
http://link-to-my-share-helper.com/?sharedUrl=https%3A%2F%2Fdeezer.page.link%2FebTURw42M8zrrRSk6
